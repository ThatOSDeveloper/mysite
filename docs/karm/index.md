# What is KARM?

KARM is a hobbyist kernel that I have been making for around a month, though it borrows parts from previous kernels, kernels that never went public, etc. Though its core design is simple, keep it stupidly designed, so stupidly designed it always fails.

I literally change everything on a moments notice for fun. May lord have mercy on your souls when it happens, because there is no Semantic Versioning, there is not even versioning, there is just commit IDs.

[KARM default distro](https://codeberg.org/KARM-Project/default) is intended to be a base distribution, something that one could make a fork of, and use and modify freely. Most code licensed under GPLv3 (except code from external projects, to be sure on any external licenses please check kernel/LICENSES for any external code in the kernel, the 'default' repo has no external code)
